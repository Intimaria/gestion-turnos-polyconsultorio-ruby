# README


## Ruby version
  
ruby 2.7.4p191

## System dependencies
```
yarn 
npm 
rbenv
ruby 
bundler
```
## Configuration
```
git clone https://github.com/Intimaria/turnos-polycon.git polycon
cd polycon
bundle intall
yarn install
```

## Database creation
```
rake db:migrate
```
## Database initialization
```
rake db:seed
```
## Run
```
rails s
```
visit 
localhost:3000 in your browser
